# Memory Game – WPF / C#

Ett Memory-spel utvecklat i C# och WPF enligt MVVM-arkitektur.  
Projektet skapades som en del av en kurs i systemutveckling med fokus på
arkitektur, samarbete och ren, underhållbar kod.

Arbetet genomfördes i team med ett agilt arbetssätt och GitHub för
versionshantering och samarbete.

## Funktioner
- Memory-spel där spelare turas om lokalt
- Poängräkning med enkel highscore
- Användarvänligt gränssnitt byggt i XAML
- Tydlig separering mellan logik och presentation (MVVM)
- Bakgrundsmusik och ljudeffekter (SFX)
- Centraliserad ljudhantering via Audio Service
- Volym- och mute-kontroll för musik och ljudeffekter

## Tekniker
- C#
- WPF (Windows Presentation Foundation)
- MVVM (Model–View–ViewModel)
- XAML & Data Binding
- Git & GitHub (Issues, branches, Pull Requests)
- PostgreSQL (lagring av resultat/highscore)
- SQL-frågor för att hämta och registrera nytt spelresultat.
- Interface-baserad ljudhantering med tydlig separation av ansvar


## Arbetssätt
- Agilt arbetssätt (Scrum-inspirerat)
- Arbete med user stories och acceptanskriterier
- Code reviews via Pull Requests
- Clean Code-principer och kontinuerlig refaktorering
